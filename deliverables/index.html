Welcome to the site!

This is where I am storing all of the deliverables for my capstone!

Hovering over the icons on the right will show you a few links that you can click. Clicking on "Proposal" will show you the project proposal I wrote for this project. Tapping on "Tracker" will pull up a list of dates and a quick message about what I did that day. Banging on "Write-Up" will display my final write-up for this project. The last two options are a bit different, clicking on either of them will cause you to leave this site! If you smash the "Project" button it will redirect you to the actual website I created in React. Finally, flicking the "GitHub" option will send you over to the GitHub repository for this project. This contains all the files I created in the process of doing this project.

This project will be an exercise in creating an appealing user interface and user experience. It will also be used for experimenting with technologies such as React and Docker. Finally, the end goal is to create a modular platform to easily host and show-off any other applications that are created in the future.

Technologies:
  • HTML5
  • CSS3
  • JavaScript
  • React
  • Docker
  • Vultr
  • NGINX

    Working Goals:
  • Install NPM and Docker.
  • Initialize a React application using create-react-app.
  • Create a visually appealing React application using CSS.
  • Set up Virtual Private Server (VPS) to host the React app on.
  • Acquire a professional domain name for the VPS.
  • Install NGINX and set up SSL certificates.
  • Containerize the React application using Docker.
  • Move the Docker container to the VPS.
  • Spin up the Docker container and start hosting the React app publicly.
  • Create a modular platform to easily host and show-off future projects.

    Extended Goals:
  • Containerize this capstone site.
  • Set up a capstone site to run on a subdomain through the VPS.
  • Establish an SSL certificate for the capstone subdomain.
  • Create another project to display on the main site.
  • Containerize the new project and host it on the VPS.
  • Containerize the Nginx server itself for even more modularity and isolation.
